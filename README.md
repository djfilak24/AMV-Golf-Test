# AMV Golf

Metal products made for the Fairway.

## Stack
- Next.js 14 (App Router)
- React 18
- Deployed on Vercel

## Local dev
```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)

## Deploy
Connected to Vercel via GitHub. Push to `main` → auto-deploys.

## Adding pages
Create new files in `/app` — e.g. `app/products/page.jsx` for `/products` route.

## Adding images
Drop files in `/public/images/` and reference as `/images/filename.jpg`
